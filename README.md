## Features

*   **Vertical Feed**: Smooth snap-scrolling videos like TikTok.
*   **AI Studio**: Generate videos and captions using Gemini.
*   **Live Calls**: Voice and Video call interface.
*   **Admin Panel**: Manage users and monetization.
*   **Bangla Support**: Localized interface for Bangladeshi users.
